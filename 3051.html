１. 循环
基本上跟java里的循环是一样的。
for loop
for (var=startvalue;var<=endvalue;var=var+increment)
{
    code to be executed
}

while loop
while (var<=endvalue)
{
    code to be executed
}

do...while loop
do
{
    code to be executed
}
while (var<=endvalue);
