转：两种方法解决URL中的中文参数问题
2008-03-09 16:44:01 

假设请求的URL中问号后的字符串是："country=中国"

① String类的构造方法

   1. String s = null;   
   2. try {   
   3.      s = new String(request.getParameter("country").getBytes("ISO-8859-1"), "GBK");   
   4. } catch (UnsupportedEncodingException e) {   
   5.      e.printStackTrace();   
   6. }  

② java.net包里的URLEncoder和URLDecoder

   1. String m = null;   
   2. String n = null;   
   3. try {   
   4.      m = URLEncoder.encode(request.getParameter("country"), "ISO-8859-1"); //%D6%D0%B9%FA   
   5.      n = URLDecoder.decode(m, "GBK");  // 中国   
   6. } catch (UnsupportedEncodingException e) {   
   7.      e.printStackTrace();   
   8. }
