Microsoft Windows is a user-friendly Operating System, but it lacks of some powerful command-line programs. In this post, I will recommand some command-line programs.


0. wget http://www.gnu.org/software/wget/

GNU Wget is a free software    package for retrieving files using HTTP, HTTPS and FTP, the most widely-used   Internet protocols.


1. vim  http://www.vim.org/

vim is most powerful text editor in the world.


2. upx http://upx.sourceforge.net/

upx can compress programs to make them smaller. However, the compressed program can be excuted without the need of decompressing.


3. 7z http://www.7-zip.org/

7z is the new archive format, providing high compression ratio.


4. gzip http://www.gnu.org/software/gzip/

gzip can compress files on the fly, but the compression ratio is not high.


5. psftp http://www.chiark.greenend.org.uk/~sgtatham/putty/

This is part of putty suit. psftp is a sftp client, similar to ftp.
