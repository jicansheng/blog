Speed Up Multiple SSH Connections to the Same Server
2008-12-28 14:43:31 

Add the following 3 lines in ~/.ssh/conifg(if not exists, create one yourself):

Host *
   ControlMaster auto
   ControlPath ~/.ssh/master-%r@%h:%p

By doing this, you won't have to type the password the second time to login to the same server.

See more information by typing "man ssh_config" in the terminal.
